Driver Drowsiness Detection

  • In this project, we will be making a Drowsiness Detection System.
  • Drivers and people's travelling long distance so, it suffer from lack of sleep.
  • So prevent these accidents we will builded a system using python, OpenCV, Imutils, Pygame which will alert the driver when he feels sleepy.
  • This technology prevent accidents that are caused by drivers who fell asleep while driving.
  • The Objective of this intermidiate python project is to build a Drowsiness Detection System that will detect that a person's eyes are closed.
  • This system will alert the driver where drowsiness is detected.
